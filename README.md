# Meu Portfólio

Meu portfólio pessoal como desenvolvedor, criado para apresentar um pouco sobre mim, meus projetos, serviços e formas de contato.

## Sobre o projeto

Esse projeto foi feito para praticar HTML e CSS criando uma página mais completa, com apresentação, cards de projetos, seção de serviços e links de contato.

Também usei ele para colocar em prática minha evolução nos estudos e deixar meus projetos organizados em um só lugar.

## Tecnologias usadas

- HTML
- CSS
- Git
- GitHub

## O que pratiquei

- Estrutura de landing page
- Organização de seções
- Cards de projetos
- Links externos
- Estilização com CSS
- Organização de arquivos
